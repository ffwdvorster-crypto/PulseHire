# PulseHire Starter

This is a minimal starter for PulseHire with login + theme + logo.

## Default Login
- Email: `admin@pulsehire.local`
- Password: `admin`

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud
- Push to GitHub
- Set entry point: `app.py`
- Done!
